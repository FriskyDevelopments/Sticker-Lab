# Frisky Developments Ecosystem

This repository documents the brand and product system for the Frisky Developments ecosystem.

## Brand architecture
- **Frisky Developments** is the parent creative studio and personality layer. It is the signature behind the ecosystem: warm, generous, playful, and slightly mischievous.
- **STIX MΛGIC** is the expressive engine. It is the magic itself: the system that turns interaction into an animated, responsive experience.
- **LORE** is the identity layer. It gives the magic a profile, memory, and user-facing presence.

In short:
- **Frisky gives the magic**
- **STIX is the magic**
- **LORE gives that magic identity**

## What this repo now covers
- parent-brand positioning for Frisky Developments
- ecosystem brand architecture across Frisky, STIX MΛGIC, and LORE
- voice and tone guidance for collaborators
- practical logo direction for the parent mark
- system notes for STIX implementation work

## Current state of the repo
At the moment, the repository is documentation-first. The existing STIX handoff established product and motion-system direction, but it did not clearly define the parent brand above STIX or explain how LORE fits into the same system. The new brand documents are intended to close that gap and give future design, product, and implementation work a consistent strategic frame.
